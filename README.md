# Nischal Pantha's Portfolio - Pilot & NamesTech Founder

A personal portfolio website for Nischal, a pilot and founder of NamesTech, showcasing the Skybound app for pilot training and operations.

## Setup

1. Install Python 3.x if not already installed.
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Install Node.js and npm if not already installed.
4. Install npm dependencies:
   ```
   npm install
   ```
5. Run the app:
   ```
   python src/app.py
   ```
6. Open your browser to http://127.0.0.1:5000/

## Required Pictures

Place the following images in the `static/images/` directory:

- `profile.jpg`: Your profile picture (recommended: 300x300px, square format)
- `app_screenshot.jpg`: Screenshot of the Namestech app interface
- `background.jpg`: Background image for the header (optional, 1920x1080px)
- `project1.jpg`, `project2.jpg`, etc.: Additional project images if needed

## Features

- Smooth animations and transitions using Anime.js
- Responsive design
- Sections for about, projects, features, and contact
- Ready for customization with your content
